# ENTREGA 09 – Lista de Requisitos de Infraestrutura

Este documento lista as tecnologias e versões utilizadas para o desenvolvimento e hospedagem do sistema.

| Item | Especificação | Versão (Mínima/Utilizada) |
|:---|:---|:---|
| **SGBD** | MySQL / MariaDB (via XAMPP) | 10.4.x ou superior |
| **Linguagem Backend** | PHP (com extensão MySQLi) | 8.0.x ou superior |
| **Linguagem Frontend** | HTML5 / Vanilla CSS / Vanilla JavaScript | ES6+ |
| **Servidor Web** | Apache HTTP Server | 2.4.x |
| **Sistema Operacional** | Windows | 10 / 11 |
| **Navegador** | Google Chrome / Microsoft Edge | Última versão estável |
